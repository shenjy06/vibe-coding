# Camera & Tab Recorder

Chrome 浏览器插件，支持摄像头录像和浏览器标签页录制，摄像头画面支持正方形 / 圆形显示。

## 功能

- **摄像头预览**：实时显示摄像头画面，支持正方形和圆形两种形状切换
- **可调大小**：拖动滑块 100px ~ 600px 实时调整预览窗口尺寸
- **摄像头录制**：录制摄像头画面（含音频），录制的是渲染后的画面（带形状裁剪效果）
- **拍照截图**：一键截取当前摄像头画面保存为 PNG
- **Tab 录制**：录制浏览器标签页内容（含音频），自动下载 WebM 视频

## 安装

### 方式一：加载解压扩展（开发模式）

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目 `chrome-record` 文件夹

### 方式二：加载 .crx 文件

1. 打开 `chrome://extensions/`，开启开发者模式
2. 将 `chrome-record.crx` 文件拖入页面即可安装

> 首次打包需在 `chrome://extensions/` 点击「打包扩展程序」生成 `.crx` 和 `.pem` 文件。

## 使用

点击浏览器工具栏的插件图标，弹出控制面板：

| 按钮 | 功能 |
|------|------|
| 打开摄像头窗口 | 弹出独立窗口显示摄像头画面 |
| 开始录制 Tab | 选择标签页并开始录制 |
| 开始录制摄像头 | 打开摄像头窗口并自动开始录制 |

### 摄像头窗口

- **形状切换**：点击工具栏「正方形 / 圆形」按钮
- **大小调节**：拖动滑块调整画布尺寸
- **录制**：点击红色圆形按钮开始 / 停止
- **截图**：点击「拍照」按钮保存当前画面

### Tab 录制

1. 点击「开始录制 Tab」
2. 在弹出的系统选择器中选择要录制的标签页
3. 录制过程中可实时预览
4. 点击「停止录制」或浏览器地址栏旁的「停止共享」结束录制
5. 录制完成后自动下载 WebM 视频

## 项目结构

```
chrome-record/
├── manifest.json        # Chrome Manifest V3 配置
├── background.js        # 后台 Service Worker
├── popup.html/css/js    # 插件弹窗控制面板
├── camera.html/css/js   # 摄像头预览窗口
├── recorder.html/css/js # Tab 录制窗口
└── icons/               # 插件图标（16/48/128px）
```

## 技术栈

| 技术 | 用途 |
|------|------|
| Chrome Manifest V3 | 插件清单格式 |
| `getUserMedia` | 获取摄像头和麦克风流 |
| `getDisplayMedia` | 选择并捕获标签页画面 |
| `MediaRecorder` | 录制媒体流 |
| Canvas 2D | 实时渲染 + 圆形/方形裁剪 |
| `captureStream()` | 从 Canvas 捕获画面流 |
| Service Worker | 后台消息通信 |

## 浏览器兼容性

- Chrome 88+
- Edge 88+（Chromium 内核）
- 其他 Chromium 系浏览器

## 注意事项

- Tab 录制使用 `getDisplayMedia` API，会弹出系统原生选择器让用户选择要录制的标签页
- 录制格式为 WebM（VP9/Opus 编码），Chrome 原生支持播放
- `.pem` 私钥文件是更新插件的关键，请妥善保管，不要提交到版本库
