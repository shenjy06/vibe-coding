# 证件照制作工具

纯前端证件照制作工具，无需服务器，直接在浏览器中完成证件照制作。

## 功能特性

- 拖拽/点击上传照片
- 多种证件照尺寸预设（一寸、二寸、护照等）
- 自定义尺寸支持
- 背景颜色替换（白、蓝、红、自定义）
- 背景擦除/恢复工具
- 亮度、对比度调整
- 图片缩放
- 可拖拽裁剪框
- 排版打印功能（A4纸）
- 裁剪线显示/隐藏
- 单张下载 / 排版下载

## 操作说明

### 上传照片
- 点击上传区域选择照片
- 或直接拖拽照片到上传区域
- 建议使用正面免冠照片

### 选择尺寸
- 预设尺寸：一寸、二寸、小一寸、小二寸、护照
- 自定义：输入宽度和高度（毫米）

### 背景处理
- 点击颜色按钮更换背景色
- 使用"擦除背景"工具去除原背景
- 使用"恢复原图"工具恢复被擦除的部分

### 调整
- 亮度：调整图片明暗
- 对比度：调整图片对比度
- 缩放：调整人像大小

### 生成与下载
- 点击"生成证件照"预览效果
- 下载单张证件照
- 选择排版布局（1寸×8、2寸×4）
- 下载排版后的A4页面

## 运行方式

直接在浏览器中打开 `index.html`：

```bash
# Windows
start index.html

# 或用 HTTP 服务器
python -m http.server 8000
```

## 技术栈

- HTML5
- CSS3（渐变、Grid 布局、Backdrop Filter）
- JavaScript（Canvas API、File API）
- 零外部依赖
