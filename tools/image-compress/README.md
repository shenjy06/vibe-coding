# 图片压缩工具

纯前端图片压缩工具，无需服务器，直接在浏览器中完成图片压缩和尺寸调整。

## 功能特性

- 拖拽/点击上传图片
- 压缩质量实时调节（1-100%）
- 尺寸调整联动（锁定宽高比）
- 支持 JPG/PNG/WebP 输出格式
- 实时预览压缩效果
- 文件大小对比统计
- 一键下载压缩图片

## 操作说明

### 上传图片
- 点击上传区域选择图片
- 或直接拖拽图片到上传区域

### 压缩设置
- **压缩质量**：拖动滑块调整，数值越高质量越好但文件越大
- **输出格式**：选择 JPG、PNG 或 WebP
- **调整尺寸**：
  - 输入宽度自动计算高度（保持宽高比）
  - 输入高度自动计算宽度（保持宽高比）
  - 点击 🔒/🔓 按钮切换锁定/解锁宽高比
  - 点击"重置尺寸"恢复原始大小

### 下载
- 点击"下载压缩图片"保存到本地

## 运行方式

直接在浏览器中打开 `index.html`：

```bash
# Windows
start index.html

# 或用 HTTP 服务器
python -m http.server 8000
```
