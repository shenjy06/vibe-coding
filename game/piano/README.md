# Virtual Piano

一个基于 Web Audio API 的浏览器虚拟钢琴，纯前端实现，无需任何外部依赖。

**在线体验**：https://piano.shenjy.workers.dev/

## 特性

- **加法合成引擎** — 基频 + 3 次谐波叠加，带自然衰减曲线，模拟真实钢琴音色
- **低通滤波器** — 温暖音色，随时间频率渐降
- **多输入方式** — 支持鼠标点击、触摸屏和键盘演奏
- **零依赖** — 单个 HTML 文件，直接打开即可使用

## 音域

覆盖 2 个半八度（C3 ~ E5），共 17 个白键 + 10 个黑键。

## 键盘映射

| 键盘 | A | S | D | F | G | H | J | K | L | ; | ' | Z | X | C | V | B | N |
|------|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 音符 | C3 | D3 | E3 | F3 | G3 | A3 | B3 | C4 | D4 | E4 | F4 | G4 | A4 | B4 | C5 | D5 | E5 |

### 黑键
| 键盘 | W | E | T | Y | U | O | P | ] | [ | M |
|------|---|---|---|---|---|---|---|---|---|---|
| 音符 | C#3 | D#3 | F#3 | G#3 | A#3 | C#4 | D#4 | F#4 | G#4 | A#4 |

## 使用

直接在浏览器中打开 `index.html` 即可。支持桌面和移动端。

```bash
# 或者用任意 HTTP 服务器启动
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 技术栈

- **HTML / CSS / JavaScript** — 纯前端，无框架
- **Web Audio API** — 音频合成与播放
- **加法合成（Additive Synthesis）** — 多谐波叠加生成钢琴音色
