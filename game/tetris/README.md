# 俄罗斯方块

一个使用纯 HTML/CSS/JavaScript 实现的经典俄罗斯方块游戏，无需任何外部依赖。

## 游戏特性

- 7 种经典方块（I、O、T、S、Z、J、L）
- 幽灵方块预览（显示落点位置）
- 下一个方块预览
- 分数、等级、消行统计
- 等级加速（等级越高下落越快）
- 暂停/继续功能
- 游戏结束与重新开始

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转 |
| ↓ | 加速下落 |
| Space | 硬降（直接落到底部） |
| P | 暂停/继续 |

## 计分规则

| 消行数 | 基础得分 |
|--------|----------|
| 1 行 | 100 |
| 2 行 | 300 |
| 3 行 | 500 |
| 4 行 | 800 |

最终得分 = 基础得分 × 当前等级。每消 10 行升一级，下落速度递增。

## 运行方式

直接在浏览器中打开 `index.html` 即可游玩，无需安装任何依赖或启动服务器。

```
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

## 技术栈

- HTML5 Canvas
- CSS3
- JavaScript（ES6+）
- 无第三方依赖
